### 二、 Wigner 小 d 矩阵的解析推导

Wigner 小 d 矩阵 $d^l_{m', m}(\beta)$ 的通用定义公式极为复杂，由 Wigner 本人给出：

$$d^l_{m', m}(\beta) = \sqrt{(l+m')!(l-m')!(l+m)!(l-m)!} \sum_{k} \frac{(-1)^{k-m+m'} (\cos\frac{\beta}{2})^{2l-2k+m-m'} (\sin\frac{\beta}{2})^{2k-m+m'}}{k! (l+m-k)! (l-m'-k)! (k-m+m')!}$$

**第一步：代入赤道条件 $\beta = \pi/2$**

当 $\beta = \pi/2$ 时，三角函数项变得极其优美：

$$\cos\frac{\pi}{4} = \sin\frac{\pi}{4} = \frac{1}{\sqrt{2}}$$

代入后，所有的三角函数幂次相加恰好抵消了复杂的依赖，变成了全局的常数因子：

$$\left(\frac{1}{\sqrt{2}}\right)^{2l-2k+m-m'} \left(\frac{1}{\sqrt{2}}\right)^{2k-m+m'} = \left(\frac{1}{\sqrt{2}}\right)^{2l} = 2^{-l}$$

**第二步：代入引力波的自旋权重 $m_{\text{inc}} = 2$**

为了避免符号混淆，我们设黑洞坐标系的角动量投影为 $m$，将入射自旋态设为 $2$。代入上式（即令 $m' \to m$，且原本式子中的 $m \to 2$）：

$$d^l_{m, 2}(\pi/2) = \frac{\sqrt{(l+m)!(l-m)!(l+2)!(l-2)!}}{2^l} \sum_{k=k_{\min}}^{k_{\max}} \frac{(-1)^{k-2+m}}{k! (l+2-k)! (l-m-k)! (k-2+m)!}$$

其中，求和指标 $k$ 必须保证分母中所有的阶乘变量非负（$x! \ge 0 \Rightarrow x \ge 0$），这就给出了 $k$ 的截断范围：

- $k \ge 0$
    
- $k \ge 2 - m$
    
- $k \le l + 2$
    
- $k \le l - m$
    

综上，求和的上下限被严格卡死在：

$$k_{\min} = \max(0, 2-m)$$

$$k_{\max} = \min(l+2, l-m)$$

既然理论已经打通，现在需要将其落地为代码。请在您的 `physics.py` 或新建的 `wigner.py` 中实现以下高精度函数：

Python

```
import mpmath as mp

# 开启高精度，阶乘计算在 l>20 时极易溢出
mp.mp.dps = 50 

def wigner_d_pi_half(l, m):
    """
    计算 d^l_{m, 2}(pi/2) 的解析系数
    仅需传入 l 和 m。内部利用对称性处理 m < 0 的情况。
    """
    if abs(m) > l:
        return mp.mpf('0.0')
        
    # 利用对称性规避负 m 的计算
    if m < 0:
        sign = mp.power(-1, l + m)
        return sign * wigner_d_pi_half(l, -m)
        
    # 提取公共前置系数
    prefactor = mp.sqrt(mp.fac(l + m) * mp.fac(l - m) * mp.fac(l + 2) * mp.fac(l - 2))
    prefactor = prefactor / mp.power(2, l)
    
    # 确定求和边界
    k_min = max(0, 2 - m)
    k_max = min(l + 2, l - m)
    
    sum_term = mp.mpf('0.0')
    for k in range(k_min, k_max + 1):
        numerator = mp.power(-1, k - 2 + m)
        denominator = mp.fac(k) * mp.fac(l + 2 - k) * mp.fac(l - m - k) * mp.fac(k - 2 + m)
        sum_term += numerator / denominator
        
    return prefactor * sum_term
```
